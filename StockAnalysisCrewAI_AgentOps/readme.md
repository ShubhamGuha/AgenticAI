# Stock Analysis with CrewAI and AgentOps

 This Jupyter notebook demonstrates a multi-agent system for stock analysis
 using CrewAI, integrated with AgentOps for enhanced observability and monitoring.
 The system utilizes specialized agents to gather financial news, analyze stock
 price trends, and generate investor-friendly reports.

# Key Components:
 - **CrewAI:** A framework for building multi-agent systems.
 - **AgentOps:** An observability platform for AI agents, providing insights into agent
   executions, tool usage, and task management.
 - **Gemini LLM:** Utilized as the underlying large language model for agent intelligence.
 - **Custom Tools:**
   - `StockSearchTool`: Leverages DuckDuckGo for fetching real-time stock news.
   - `YahooFinanceTool`: Integrates with yFinance to retrieve historical stock price data.

# Workflow:
 1. **Environment Setup:** Loads API keys for Gemini and AgentOps.
 2. **Tool Definition:** Custom tools are defined for news searching and financial data fetching.
 3. **Agent Configuration:** Two agents, a 'Stock Analyst' and a 'Report Generator', are
    instantiated with specific roles, goals, and backstories.
 4. **Task Creation:** Tasks are defined for news gathering, trend analysis, and report
    writing, assigning them to the appropriate agents and tools.
 5. **Crew Assembly:** The agents and tasks are assembled into a CrewAI crew.
 6. **Execution:** The crew is kicked off to perform the stock analysis, with AgentOps
    logging all agent activities.
 7. **Output:** The final investor report generated by the 'Report Generator' agent is printed.

 This implementation showcases how to build intelligent, observable agent systems for
 complex tasks like financial analysis.

## Setup

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt` (assuming you have a requirements.txt).
3. Set up environment variables:
   - `GEMINI_API_KEY`: Your API key for the Gemini LLM.
   - `AGENTOPS_API_KEY`: Your API key for AgentOps.
   (The script will prompt you if these are not set).

## Usage

Run the Jupyter notebook:
`jupyter notebook AgentOps_Practical_Implementation.ipynb`

Or execute the Python script:
`python AgentOps_Practical_Implementation.ipynb`

The script will output a final stock analysis report for 'AAPL'.
